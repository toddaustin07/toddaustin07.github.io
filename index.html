To get started with enabling your Raspberry Pi to run SmartThings direct connect device applications:

1) Read the 1-page Quickstart document

2) Download this file to your Pi's home directory: mastersetup

Right-click on link above; OR...

From your Pi home directory: pi@raspberrpi:~ $ wget http://toddaustin07.github.io/mastersetup

3) pi@raspberrypi:~ $ chmod +x mastersetup

4) pi@raspberrypi:~ $ ./mastersetup



Note: Do-it-yourselfers that want a step-by-step manual description can reference this document.
